# 🚀 Enterprise Multi-Modal Knowledge Engine (RAG+)

### 💡 Решаемая проблема (Problem Statement)
В условиях «полевой» работы (склады, логистика, аудит недвижимости, сложные путешествия) сотрудники часто ограничены в использовании ПК для поиска в корпоративных базах знаний. Поиск по PDF-инструкциям или Wiki-страницам вручную занимает слишком много времени. 

**Данный проект решает проблему мгновенного доступа к экспертным знаниям через любой тип входных данных: голос, фото или текст.**

### 💎 Бизнес-ценность (Business Value)
* **Снижение операционных затрат (OpEx):** интеллектуальное кэширование (**L1 MD5 Cache**) снижает расходы на API и вычислительные мощности на **40-60%**.
* **Минимизация человеческого фактора:** оценка точности через **Ragas (0.80+)** гарантирует, что система выдает ответы строго на основе верифицированных регламентов, исключая галлюцинации.
* **Hands-free эффективность:** сотрудник получает информацию через голос или фото, не прерывая основной рабочий процесс.

### 🏗 Архитектура системы (Architecture & Implementation)
Проект представляет собой **Domain-Agnostic Framework** (доменно-независимый каркас). В качестве демонстрационного кейса (**Proof of Concept**) выбрана система навигации и поддержки в сложной неструктурированной среде (Шри-Ланка).

#### 1. Многоуровневый Retrieval Pipeline
Система работает по принципу «умного каскада»:
* **⚡ L1 (Efficiency):** мгновенная выдача из кэша по хэшу контекста (MD5).
* **🧠 L2 (Expertise):** семантический поиск по векторной базе знаний (**ChromaDB**) с глубоким анализом контекста.
* **🌐 L3 (Real-time):** динамический вызов **Tavily API**, если локальных данных недостаточно для точного ответа.

#### 2. Мультимодальные сценарии (Multi-modal Use Cases)
* **Vision-аналитика:** автоматический скоринг объектов. *Пример:* Анализ фото помещений на соответствие ТЗ (наличие рабочих зон, инфраструктуры).
* **Voice-to-Knowledge:** мгновенная транскрипция запроса в поисковый вектор. *Пример:* Запрос юридических правил или логистики «на ходу» через **Whisper API**.
* **Объектное распознавание:** идентификация неизвестных объектов и мгновенное сопоставление их с локальной базой знаний для выдачи спецификаций.

### 📊 Метрики качества и надежности
Система прошла автоматизированную валидацию качества генерации (QA):
* **Faithfulness (`0.80`):** высокий уровень верности источнику данных.
* **Answer Relevancy (`0.95`):** точное соответствие ответа бизнес-запросу пользователя.
* **State Management:** удержание контекста диалога на глубине **20+ шагов**, что позволяет решать сложные многоуровневые задачи.

### 💻 Стек технологий (Tech Stack)
* **Engine:** Python | LangChain | OpenAI (GPT-4o Vision/Whisper)
* **Storage:** ChromaDB (Vector Store) | LRU-based JSON Cache
* **Interface:** Aiogram 3.x (Production-ready Telegram Framework)
* **QA & Analytics:** Ragas | Pandas | Datasets

### 🛡 Безопасность и масштабируемость
* **Production Config:** полное соответствие методологии **The Twelve-Factor App** (изоляция секретов через переменные окружения).
* **Scalability:** архитектура позволяет адаптировать систему под любую индустрию (LegalTech, FinTech, Medicine) путем простой замены директории `/data`.
